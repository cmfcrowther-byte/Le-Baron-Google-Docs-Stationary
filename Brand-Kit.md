# Le Baron — Brand Kit

## Palette

| Name      | Hex       | Use |
|-----------|-----------|-----|
| Gold      | `#D4AF37` | Accents, links, highlights |
| Off-white | `#f1f0ea` | Backgrounds, light surfaces |
| Charcoal  | `#1A1A1A` | Primary text, headings |
| Stone     | `#877b64` | Secondary text, borders |
| Midnight  | `#00091d` | Dark backgrounds, footer |

## Fonts

- **Serif:** Cormorant Garamond  
- **Sans:** Inter  

## Google Docs — Adding Fonts

Go to **Font Menu** → **More Fonts** → Search for **Cormorant Garamond** and **Inter** to add them to your document.
