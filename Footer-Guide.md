## 3-step styling guide (Google Docs)

### Step 1 — Font size
- Select the footer text.
- Go to **Format** → **Text** → **Size** (or use the font size dropdown in the toolbar).
- Set the size to **7pt** or **8pt** (8pt is easier to read; 7pt looks more discreet).

### Step 2 — Colour
- With the footer text still selected, go to **Format** → **Text** → **Colour**.
- Choose **Custom** and enter: **#877b64** (Stone from your brand palette).
- Click **OK**.

### Step 3 — Thin top border (separate footer from body)
- Double-click inside the **footer area** (bottom of the page) to focus it.
- Go to **Format** → **Paragraph styles** → **Border and shading** (or **Format** → **Paragraph** and look for borders).
- In Google Docs: **Format** → **Paragraph styles** → **Options** → **Border and shading**.
- Add a **top border** only: style **thin** (e.g. 1pt), colour **#877b64** or **#1A1A1A** to match the footer.
- Apply. The line will sit above the footer text and separate it from the body.

**Tip:** If your version uses **Insert** → **Header and footer** → **Footer**, the border is often applied to the footer paragraph: select that paragraph, then use **Format** → **Paragraph** (or **Borders and lines**) and set a top border.
